This easy-to-use plugin allows WooCommerce store owners to view the most recent WooCommerce orders on WordPress dashboard. 

== Description ==

This plugin allows WooCommerce store owner to view most recent orders on WordPress dashboard. They can view, process and complete the order and also it show the current order status and allows to visit the Order Edit page of that order.
== Installation ==

Method 1:
<ol>
<li> Download the plugin wodw-orders-dashboard-widget.zip </li>
<li> Unpack the wodw-orders-dashboard-widget.zip file and extract the wodw-orders-dashboard-widget folder </li>
<li> Upload the plugin folder to your /wp-content/plugins/ directory </li>
<li> Go to WordPress dashboard, click on Plugins from the menu </li>
<li> Locate the Woocommerce Orders Dashboard Widget and click on Activate link to activate the plugin. </li>
</ol>

Method 2:
<ol>
<li> Download the plugin wodw-orders-dashboard-widget.zip </li>
<li> Go to WordPress Dashboard-Plugins-Add New </li>
<li> Click on Upload Plugin link from top </li>
<li> Upload the downloaded wodw-orders-dashboard-widget.zip file and click on Install Now </li>
<li> After installation, click on Activate Plugin link to activate the plugin. </li>
</ol>
